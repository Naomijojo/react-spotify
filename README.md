# 專案名稱：React Spotify

## 專案說明
React Spotify 是一個模擬Spotify的音樂播器，提供播放、暫停、搜尋、收藏歌曲等功能。

## 專案技術
- 前端框架: React + React router + zustand
- 樣式管理: CSS + Tailwind 
- UI框架: Ant design
- 打包工具: Vite

## 專案功能
1. 使用JWT實現token-based登入、註冊驗證機制
2. 實現使用者操作後的回饋機制(例登入成功message)
3. 播放 / 暫停 / 收藏 / 循環 / 搜尋 歌曲

## 下載專案
```
git clone https://github.com/Naomijojo/react-spotify.git
```
## 進入專案目錄
```
cd react-spotify
```
## 安裝依賴
```
npm install
```
## 啟動專案
```
npm run dev
```
